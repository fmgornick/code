class  FilterTest
 This is a series of unit tests to check how accurate my FFT image convolution filter works. In the image directory, there are a series of images with "expected" and "actual" suffixes. The expected comes from an online gaussian blur filter with size 13 kernel and an (i just guessed) standard deviation of 2. The site is https://pinetools.com/blur-image. The actual output comes from our image_processor software, and I check to see how similar the pixel values are by summming up the differences and dividing by the size of the image. More...
