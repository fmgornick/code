This is the function that does all the logic work, it takes the processesd images and returns true of false if the robot is inside the images. It then returns the X and Y coordinates of where the robot is in the image. This information is stored in the top left pixel of the blob image. The format is (<1 or 0, can see>, <robot x>="">, <robot y>="">).
