Detailed Description

The main class for NonMaxFilter, extends Filter. NonMax takes in two images, an intensity image and a direction image. This filter scans the image and if intensity of the nearest pixel in the given direction is larger than the current pixel, it will set the current pixel to black. The end result is an image that has thinner lines.
