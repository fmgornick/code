Detailed Description

The main class for SimpleFilter, extends Filter. This filter does the brunt of the work when iterating through an image. It's an abstract class, so it's initialized as one of it's derived classes, so when it iterates through the image, it calls the ApplyAtPixel() function that's defined in each of it's derived classes.
